Biodata website — Jeffery

How to view

- Open `index.html` in your browser (double-click) or run in cmd:

```
start "" "index.html"
```

- Or use Live Server in VS Code for auto-reload.

Files created

- `index.html` — main biodata page
- `contact.html` — contact details page
- `styles.css` — shared transparent UI styles
 - `apple.html` — Apple-style liquid translucent biodata page
 - `apple.css` — styles for the Apple-like page

Customizing

- Replace the placeholder image by editing the `img` `src` attribute in `index.html`.
- Update name, role, contact, skills and sections directly in `index.html`.
- `styles.css` contains the visual variables and layout; tweak `:root` values for colors and radii.

Notes

- The site is static and responsive with glass/transparent cards. To publish, upload the folder to any static host (GitHub Pages, Netlify, Vercel).
